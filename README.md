# ceb_simplify_frames
Blender tool to help on mocap cleaning

Using this Blender addon, you can clean keyframes fopr objects, for bones and even on specific fcurves.

It works by taking in consideration the frames that you mark (using the "M" key on timeline or graph editor.)
